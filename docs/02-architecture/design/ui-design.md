# Dashboard Design

The gold layer is built to answer these questions. Each question maps to data the dashboard needs to display.
Add questions here first — then figure out what tables/columns are needed to answer them.

---

## Portfolio View

Questions the portfolio page should answer:

- What is my total portfolio value right now?
- What is my total return (absolute and %) since I started investing?
- How much have I invested in total (cost basis)?
- What is my portfolio return today vs. yesterday?
- Which assets are my best and worst performers?
- Which assets are dragging down the portfolio?
- How is my portfolio distributed across assets?
- What is my overall portfolio risk (volatility, beta)?
- How much cash do I have available?

---

## Asset View

Questions the asset detail page should answer:

- How much have I spent on this asset total (avg cost, cost basis)?
- What is my current return on this asset (absolute and %)?
- What is the daily return trend over time?
- What is the current drawdown — how far is it from its peak?
- Is now a good time to buy more? (price vs. avg cost, DCA opportunity)
- What is the risk of buying more right now? (volatility, drawdown)
- Is this asset undervalued or overvalued relative to its history?
- What are the moving averages showing (20d, 30d, 50d)?
- How volatile has this asset been recently?
- How does this asset correlate with the rest of my portfolio?
