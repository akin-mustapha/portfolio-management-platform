# Solo Developer Role Index

You wear all of these hats. The goal is to wear them **intentionally** — not just default to "data engineer + frontend" and skip the rest.

| Role | File | When to put it on |
|---|---|---|
| Data Engineer | [data-engineer.md](data-engineer.md) | Building or fixing pipelines, medallion layers |
| Backend Developer | [backend-developer.md](backend-developer.md) | Writing services, domain logic, calculations |
| Database Administrator | [database-administrator.md](database-administrator.md) | Schema changes, migrations, query work |
| Frontend Developer | [frontend-developer.md](frontend-developer.md) | Dashboard layout, charts, callbacks |
| Product Manager | [product-manager.md](product-manager.md) | Deciding what to build and why |
| Project Manager | [project-manager.md](project-manager.md) | Tracking work, issues, branch discipline |
| QA Engineer | [qa-engineer.md](qa-engineer.md) | Writing tests, checking coverage |
| DevOps Engineer | [devops-engineer.md](devops-engineer.md) | CI/CD, linting, dependencies |
| Tech Lead | [tech-lead.md](tech-lead.md) | Commit hygiene, PR quality, pre-commit |
| Security Engineer | [security-engineer.md](security-engineer.md) | Secrets, credentials, .env hygiene |
| Observability Engineer | [observability-engineer.md](observability-engineer.md) | Logging, monitoring, pipeline visibility |
